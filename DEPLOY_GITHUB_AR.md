# طريقة رفع GitHub V49

1. فك ضغط ملف GitHub V49.
2. ارفع محتويات الملف مباشرة في Root الريبو.
3. لازم تشوف في أول صفحة من GitHub:
   - `index.html`
   - `config.js`
   - `assets`
   - `sw.js`
   - `manifest.webmanifest`
4. لا ترفع فولدر ZIP نفسه.
5. فعل GitHub Pages على Branch `main` و Folder `/root`.
6. افتح الرابط مع:

`?v=49finalstable`

مثال:
`https://headersobhy-code.github.io/Haydar-Pack/?v=49finalstable`
